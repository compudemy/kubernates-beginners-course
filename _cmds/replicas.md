Get All Replication controller

```bash
    kubectl get replicationcontrollers
```

```bash
    kubectl get rc
```

Delete a replication controller

```bash
    kubectl delete rc <rs-name>
```

Delete a replica set

```bash
    kubectl delete rc <rs-name>
```

Get All replica sets created

```bash
    kubectl get rs
```

Delete a replica set

```bash
    kubectl delete rs <rs-name>
```
